# Manual del Encargado

## Credenciales
- Usuario: `informatica@sistema.cl`
- Clave: `encargado123`

## Funciones
- Ver eventos activos de su unidad.
- Seleccionar una card de evento o la card de visitas diarias.
- Registrar persona principal.
- Registrar vehiculo si la llegada es en vehiculo propio.
- Registrar pasajero como persona independiente.

## Inicio de sesion
1. Abrir la aplicacion.
2. Ingresar correo y clave.
3. El sistema redirige al panel del encargado.

## Ver eventos activos
- Se muestran como cards.
- Cada card abre el formulario y el listado filtrado por ese evento.

## Registrar persona
Campos principales:
- Tipo de documento
- Numero de documento
- Nombre completo
- Correo
- Telefono
- Tipo de llegada

## Registrar con vehiculo
- Seleccionar `Vehiculo propio`.
- Completar patente, marca, modelo y color.
- El sistema guarda el vehiculo asociado a la persona principal.

## Registrar con pasajero
- Marcar `Viene con pasajero`.
- Completar los datos del pasajero.
- El pasajero se guarda como persona independiente.
- El pasajero debe seleccionar su propio evento.

## Relacion con el guardia
- La persona principal y el pasajero alimentan la lista que luego revisa el guardia.
- El campo de participacion no se ingresa manualmente en la persona principal.
